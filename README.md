# ELECTRICAL PCBs

This repository contains all PCBs for UMD Loop's 2025-26 rover, Palamedes.

-----

## Guidelines
* Fork this repository for development, you can open a pull request to this repo to add your changes for everyone.
* Do NOT force push.
* Follow best practices (descriptive commit messages, well documented code, etc.)
* Add a new subdirectory for each PCB that is made

## Including Libraries
* The following libraries tab will include the Digikey, or product vendor component, CAD, Schematic, and/or Drawing for the components that are not found on KiCAD
